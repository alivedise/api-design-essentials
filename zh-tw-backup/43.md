# [ADP-43] API 限流

## 指導原則

- 應對 `audience>=public-external` 的 API 添加限流。
- [應對限流的端點添加 x-rate-limit](753.md)
- 如果超過限流，應返回與 HTTP Problem json 對齊的以下錯誤消息
    
    ```json
    POST /search HTTP/1.1
    
    HTTP/1.1 429 Too Many Requests
    
    {
      "type": "/problems/rate-limit",
      "title": "Too Many Requests",
      "status": 429,
      "detail": "You have exceeded the rate limit of 1000 requests per minute.",
      "instance": "/search",
      "retryAfter": "PT1M"
    }
    ```
