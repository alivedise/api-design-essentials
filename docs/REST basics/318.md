# [ADP-318] Query Parameter Convention

## Guidance

- SHOULD use query parameter for GET request
- SHOULD not use payload(body) for GET request
    - In browser spec, the GET request WON’T have a body(payload). See https://developer.mozilla.org/en-US/docs/Web/API/XMLHttpRequest/send
- MAY NOT use query parameter for POST request
- SHOULD use payload(body) for POST request

## Example

DO

```json
GET /events?occuredAt=2024-07-09T18:00:00Z
POST /search
{
  "occuredAt": "2024-07-09T18:00:00Z"
}
```

DON’T

```json
POST /search?occuredAt=2024-07-09T18:00:00Z
GET /events
{
  "occuredAt": "2024-07-09T18:00:00Z"
}
```
