# [ADP-309] URI format convention

## Guidance

- MUST use kebab-case in URI

## Design Thinking

This rule pertains to the naming of resource collections and the modifier/decorator to describe the collections.

<aside>
💡 When there’s a need to put the path in the property, you’ll need to transfer it to camelCase since we defined the property in JSON should be camelCase.

</aside>

## Example

```jsx
https://api.example.dev/api-principles
https://api.example.dev/api-principles/recently-visited
```
