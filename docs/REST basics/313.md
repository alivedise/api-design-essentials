# [ADP-313] Sorting

## Guidance

- SHOULD document the API support for sorting.
- SHOULD use the common query parameter `sort`.
- [DRAFT] SHOULD use `-` to represent the sorting order for descending. No sign means ascending by default.
- SHOULD use a comma-separated list to represent the ordering.

## Design Thinking

`+` in url would be encoded so we ignore it.

## Example

Get the applications list by ascending `appName` (A to Z) and then descending `releaseDate`:

```bash
GET /applications?sort=appName,-releaseDate
```
