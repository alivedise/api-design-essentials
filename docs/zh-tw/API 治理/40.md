# [ADE-40] API 治理概述

:::warning reviewing phase 1
這篇文章需要連結到更多 API 治理的 ADP。
:::

:::info
💡 本 ADE 是與 API 治理相關的 ADP 的縱覽 ADP。
:::

- API 治理的主要任務是統籌管理不同團隊和小組所擁有的 API 集合，這些團隊和小組擁有一定的自治權和獨立決策權。

- API 治理的主要目標是提高 API 景觀的價值，或者說是最大化其價值，從而提高整體的API生態系統。

- API 治理與 API 產品管理有相似之處，但它們之間的區別在於，API 治理的產品是指整個 API 景觀，而不是單一的API產品。

本 ADP 的目的是收集所有跟 API 治理相關的原則和指導方針，以確保API治理的實踐能夠遵循統一的標準和原則。
