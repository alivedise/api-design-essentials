# [ADE-708] 時區數據

## 指導原則

- 必須(MUST)使用 IANA 時區數據庫來表示格式標記為時區的字串資料。

## IANA 時區數據庫概述

IANA 時區數據庫，也稱為 tz 數據庫或 zoneinfo 數據庫，是全球公認的時區資訊標準。它提供了一套全面詳細的時區規則和歷史記錄，包括：

1. **時區名稱**：標準化的時區標識符，如 `America/New_York`、`Europe/London` 和 `Asia/Tokyo`。
2. **偏移資訊**：每個時區相對於協調世界時（UTC）的偏移詳情，包括由於夏令時和其他調整導致的歷史變化。
3. **轉換規則**：標準時間和夏令時之間轉換的規則，包括開始和結束日期。

### 示例

IANA 時區標識符的一個例子是：

```http
America/New_York
```

這代表了美國紐約的時區，包括其標準時間和夏令時規則。

示例：

```http
2024-07-08T14:30:00+05:30[Asia/Kolkata]
```
