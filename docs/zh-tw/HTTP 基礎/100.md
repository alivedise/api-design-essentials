## 最佳實踐

在使用 HTTP 時：

- 應該(SHOULD)為不同操作使用適當的 HTTP 方法。
- 必須(MUST)使用正確的狀態碼實現適當的錯誤處理。
- 應該(SHOULD)利用 HTTP 標頭進行內容協商和快取。
- 必須(MUST)使用 HTTPS 確保安全通信。
