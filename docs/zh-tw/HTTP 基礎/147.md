# [ADE-147] Accept-Patch

## 概述

這是一個僅用於回應的標頭。

## 原則

- 可以(MAY)使用 Accept-Patch 來通知客戶端，不支持使用 PATCH 方法和特定 `Content-Type` 請求的資源，同時返回 415 狀態碼。相反，如果請求標頭中未指定 Content-Type，則不應回應 Accept-Patch 或 415。

  ```http
  PATCH /planet
  Content-Type: text/example

  415
  Accept-Patch: application/vnd.planet+json
  ```

- 另一個用例是用於 `OPTIONS` 請求;通知客戶端該資源支持 PATCH 方法。
