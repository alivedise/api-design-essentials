# [ADP-751] New OpenAPI document

## Guidance

- SHOULD start from OpenAPI 3.1.0
    
    ```yaml
    openapi: 3.1.0
    ```
    
- SHOULD use yaml format for splitting purpose
    - JSON would have deeply nested object which is difficult to read.
- SHOULD move all response schema into components, see [use $ref](758.md)
