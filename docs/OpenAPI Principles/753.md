# [ADP-753] OpenAPI: x-rate-limit

## Guidance

- SHOULD add `x-limit-rate` field in rate-limited endpoints in the OpenAPI document
    
    ```json
    openapi: 3.1.0
    info:
      title: "Sample API"
      version: "1.0.0"
    paths:
      /users:
        get:
          summary: "fetch the user list"
          x-rate-limit:
            limit: 1000
            window: "PT1M"   
    ```
