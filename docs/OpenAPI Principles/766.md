# [ADP-766] OpenAPI: country code format

```yaml
openapi: 3.1.0
info:
  title: Country Code API
  version: 1.0.0
paths:
  /example:
    get:
      summary: Example endpoint
      responses:
        '200':
          description: Successful response
          content:
            application/json:
              schema:
                type: object
                properties:
                  countryCode:
                    type: string
                    format: iso-country-code
                    description: ISO 3166-1 alpha-2 country code
                    example: "US"
components:
  schemas:
    CountryCode:
      type: string
      format: iso-country-code
      description: ISO 3166-1 alpha-2 country code
      example: "US"
```
