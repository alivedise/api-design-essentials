# [ADP-759] OpenAPI: Contact info

## Guidance

- MUST provide contact info.
- MUST specify the team or business unit name in the `#/info/contact/name` field for internal API (audience <= `COMPANY_INTERNAL`)
- MUST specify the URL to the documentation location for external API (audience ≥ `PARTNER_EXTERNAL`)
- SHOULD include an email address in `#/info/contact/email`  in support purpose, for external API (audience ≥ `PARTNER_EXTERNAL`)

## Example

```yaml
openapi: 3.0.0
info:
  title: Sample API
  description: This is a sample API.
  version: 1.0.0
  contact:
    name: API Support Team
    url: https://document.site/support
    email: support@example.com
```
