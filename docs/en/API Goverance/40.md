:::warning reviewing phase 1

:::info
💡 This ADE is a category ADP for API Governance related ADPs.
:::info

- The main task of API governance is to coordinate the collection of APIs owned by different teams and groups, which have a certain degree of autonomy and independent decision-making.

- The main goal of API governance is to increase the value of the API landscape, or to maximize its value, thereby enhancing the overall API ecosystem.

- API governance is similar to API product management, but the difference between them is that the product of API governance refers to the entire API landscape, not a single API product.

The purpose of this ADP is to collect all the principles and guidelines related to API governance, to ensure that the practice of API governance can follow unified standards and principles.
