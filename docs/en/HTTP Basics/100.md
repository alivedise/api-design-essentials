# [ADE-100] HTTP Basics Overview

:::info
This ADE provides an overview of HTTP Basic ADPs.

- ADP numbering 100-109 are reserved for general HTTP info and guidance.
- ADP numbering 111-119 are reserved for HTTP Methods.
- ADP numbering 120-199 are reserved for HTTP Headers.
- ADP numbering 200-299 are reserved for HTTP Status Codes.

:::

## Key Concepts

### HTTP Methods

For detailed information on HTTP methods, refer to [ADE-111](111.md).

### HTTP Headers

For more information on HTTP headers, see [ADE-120](120.md).

### HTTP Status Codes

For a comprehensive list and usage guidelines, refer to [ADE-200](200.md).

## Best Practices

When working with HTTP:

- SHOULD use appropriate HTTP methods for different operations
- MUST implement proper error handling using correct status codes
- SHOULD leverage HTTP headers for content negotiation and caching
- MUST ensure secure communication using HTTPS
