# [ADE-750] Overview of OpenAPI Design

:::info
This ADE is an overview of the collection of specifications related to OpenAPI.
ADP number range: 750-799
:::

## OpenAPI Specification (OAS)

The OpenAPI Specification is a standard format for describing RESTful APIs. It allows both humans and computers to understand the capabilities of an API without accessing the source code or documentation.
