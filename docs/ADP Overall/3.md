# [ADP-3/D] ADP Numbering & Category

## Guidance

- Each ADP MUST have an identifier which is number now.
- ADP number: 0-1000 are reserved for general purpose ADP for HTTP API Design.
- General purpose ADP MAY define their ADP number scope.
- ADP number: 1000-2000 are reserved for general purpose ADPs in 0-1000.
- ADP number: 2000-2499 are reserved for Websocket API Design.
- ADP number: 2500-2999 are reserved for GraphQL API Design.
- ADP number: 3000-3999 are reserved for Async API Design.
- ADP number: 4000-4999 are reserved for non-HTTP based API Design.
- Application/Language/Component/Library specific ADP starts from 10000, 20000, 30000, …
