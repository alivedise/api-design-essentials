## Overview

![Untitled](%5BADP-352%5D%20Conditional%20Request%20561dda1ee45b4b2498f569d99e9903d8/Untitled.png)

## Guidance

- SHOULD provide ETag with If-None-Match and If-Match and for an API response if it is designed to be conditional.
