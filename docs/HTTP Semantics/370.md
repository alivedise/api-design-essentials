# [ADP-370] HTTP Caching: Last-Modified

## Procedures

1. The client initialized the first GET request without caching headers.
2. The server returns the data with `Last-Modified` date.
3. The client requests the GET request with `If-Modified-Since` .

## Guidance

- MAY use `Last-Modified` in the response header for caching.

## Implementation Detail

- The browser would automatically attach `If-Modified-Since` . If your client is not the browser, e.g. command line tool such as `curl` or a server side request, you may need to attach `If-Modified-Since` on your own.
- The server might decide its way to implement `Last-Modified`. For example, have a database to modify the date whenever data is changed.
