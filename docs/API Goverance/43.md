# [ADP-43] API Rate Limiting

## Guidance

- SHOULD add rate limit to `audience>=public-external` APIs.
- [SHOULD add x-rate-limit to rate-limited endpoints](753.md)
- SHOULD return the follow error message which are aligned to HTTP Problem json if the rate limit is exceeded
    
    ```json
    POST /search HTTP/1.1
    
    HTTP/1.1 429 Too Many Requests
    
    {
      "type": "/problems/rate-limit",
      "title": "Too Many Requests",
      "status": 429,
      "detail": "You have exceeded the rate limit of 1000 requests per minute.",
      "instance": "/search",
      "retryAfter": "PT1M"
    }
    ```
