# [ADP-603] CloudEvents type conventions

## Guidance

- MUST use namespaced event
- MUST reverse the domain
- MUST follow the naming convention
    
    ```json
    com.vivotek[.applicationId].resourceType.eventVersioning.eventInPastTense
    ```
    
    - The applicationId part is optional; if the resourceType part is guaranteed to be unique under the (reversed) domain, the application part could be omitted
- event versioning SHOULD follow semver
