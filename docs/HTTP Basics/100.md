# [ADP-100/O] HTTP Basics Overview

This ADP is a category overview of HTTP Basic ADPs.

- ADP numbering 100-109 are reserved for general HTTP info and guidance.
- ADP numbering 111-119 are reserved for HTTP Methods.
- ADP numbering 120-199 are reserved for HTTP Headers.
- ADP numbering 200-299 are reserved for HTTP Status.
