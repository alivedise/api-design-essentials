# [ADP-143] Expires

## Guidance

- SHOULD NOT use `Expires` since it’s not common.

## Design Thinking

- The user is able to change the system time to let `Expires` invalid.
- The browser would ignore `Expires` if `max-age` is set(as a directive in `Cache-Control`)
- `Expires` would affect `Last-Modified`.
